# 项目名称：yn-web

&emsp;&emsp;版本号：v1.0.0

&emsp;&emsp;author：Jeffery Liang

## 项目初始化

```
yarn install
```

### 编译运行开发环境

OneCare 项目

```
yarn run dev
```

聆医项目

```
yarn run dev_lingyi
```

### 编译打包正式环境代码

OneCare 项目

```
yarn run build
```

聆医项目

```
yarn run build_lingyi
```

### 编译运行测试环境

```
yarn run test
```

### 使用 eslint 进行代码格式化

```
yarn run lint
```
